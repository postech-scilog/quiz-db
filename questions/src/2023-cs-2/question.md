정지 문제(halting problem)는 임의의 컴퓨터 프로그램과 입력이 주어졌을 때 이 프로그램이 해당 입력에
대해서 영원히 작동하는지 또는 유한한 시간 내에 실행이 완료되는지 결정하는 문제이다. 이 문제의
난이도는?

1. 다항식 시간 안에 결정 가능 (P class)
2. NP-완전 (NP-complete)
3. 지수 시간 안에만 결정 가능한 것으로 알려져 있으나, NP-완전성은 알려져 있지 않음
   (Known to be solvable only in exponential time but NP-completeness unknown)
4. 결정 불가능 (Unsolvable / undecidable)

분류: **계산 이론**
