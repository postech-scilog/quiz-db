다음 주요 언어들의 특징과 발표년도를 외워두자.

* 1950s
  * Plankalkul (1943, Konrad Zuse): 최초의 고수준 언어, 설계만 됨
    * Konrad Zuse는 최초의 컴퓨터 중 하나로 꼽히는 Z3을 설계한 독일의 공학자
  * Short code (1949, John Mauchly): 최초로 구현된 고수준 언어
    * 인터프리팅되어서 동작함
    * John Mauchly는 최초의 튜링 완전한 컴퓨터인 ENIAC의 설계자
  * Autocode (1952): 최초로 컴파일되는 언어
  * FLOW-MATIC (1955): 최초로 영어 문법을 사용한 언어
  * Fortran (1957, John Backus): 최초로 널리 사용된 고수준 언어
    * 현역 언어들 중 최고령
    * John Backus는 1977년 BNF와 포트란 등의 공로로 튜링상 수상
  * COBOL (1959)
* 1960s
  * Lisp (1960, John McCarthy)
  * ALGOL 60 (1961)
    * 해당 언어 설계에 참여한 사람들 중 튜링상 수상자 다수
      (John McCarthy, John Backus, Edsger Dijkstra...)
  * BASIC (1964)
  * APL (1966, Kenneth Iverson)
    * Kenneth Iverson은 이 언어로 1979년 튜링상 수상
  * Simula (1967, Ole-Johan Dahl & Kristen Nygaard)
    * 최초의 Object-Oriented Language
    * 두 개발자는 해당 공로로 2001년 튜링상 수상
* 1970s
  * Pascal (1970, Niklaus Wirth)
    * Niklaus Wirth는 이 언어 및 MODULA 언어 등의 설계로 1984년 튜링상 수상
  * Smalltalk (1972)
    * 많은 OOP 언어들에 지대한 영향 끼침
  * C (1972, Dennis Ritchie)
    * Dennis Ritchie는 Ken Thompson과 함께 C로 UNIX를 만들었으며, 해당 공로로 1983년 튜링상 공동
      수상
  * Prolog (1972): 최초의 Logic Programming 언어
  * ML (1973, Robin Milner)
    * Robin Milner은 1991년 튜링상 수상
* 1980s
  * C++ (1983)
  * Perl (1987, Larry Wall)
* 1990s
  * Haskell (1990)
  * Python (1991)
  * Java (1995, James Goslang)
  * Javascript (1995)
    * 원래 이름은 livescript였으나 같은 연도에 발표돼 큰 인기를 얻은 Java와 비슷한 이름으로 변경
* 2000s
  * Go (2009, Google)
* 2010s
  * Rust (2010, Mozilla)
