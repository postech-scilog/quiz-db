1. 2024년 기출에서 언급했듯이 O(n^2)
2. Dijkstra's algorithm은 O(E + Vlog V) 의 시간복잡도를 가진다. Comparison sort 알고리즘과 비슷하게,
   대부분의 shortest path algorithm들의 시간 복잡도는 O(VlogV)와 비슷하다.
3. Binary search는 O(log n)
4. Boolean satisfiability 문제는 최초로 NP-Complete임이 증명된 문제이다. (Cook-Levin Theorem)
   NP-Complete에 속하는 문제들은 아직까지 알려진 알고리즘로는 exponential time이 최선이다.
