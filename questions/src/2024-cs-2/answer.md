문제에 어떤 운영체제의 규약인지 명시되어 있지 않으므로 linux라고 가정하자.

우선 x86_64 리눅스 시스템에서 **유저 공간** 함수의 호출에는 System V ADM64 ABI가 사용된다. 해당 ABI에서 사용되는 레지스터들은 다음과 같다.

1. rdi - 첫번째 인자
2. rsi - 두번째 인자
3. rdx - …
4. rcx - …
5. r8 - …
6. r9 - …

나머지 인자들은 스택으로 전달된다. 위 순서는 “**Di**ana’s **s**ilk **d**ress **c**osts **89**$” 라는 문장으로 외울 수 있다.

한편, 리눅스 커널의 시스템 콜 ABI는 조금 다르다.

1. **eax** - syscall number
2. rdi - 첫번째 인자
3. rsi - 두번째 인자
4. rdx - …
5. **r10** - …
6. r8 - …
7. r9 - …

시스템 콜은 최대 6개까지 인자를 받을 수 있다. r10 레지스터를 rcx 대신 사용한다는 점이 다르다는 것에 유의하자. 만약 문제가 “4번째 인자 전달에 사용되는 레지스터”를 물었다면 굉장히 어려운 문제가 되었을 것이다.
